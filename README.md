# EMS Dashboard

Простий односторінковий звіт складу для фракції EMS. Заповнюйте кількість виданих предметів, вводьте нік та Static у форматі `#12345`, а потім надсилайте звіт у Discord за вказаним webhook.

## Запуск

1. Відкрийте `index.html` у браузері (подвійний клік або `open index.html`).
2. Заповніть нік, Static та кількість виданих предметів.
3. Натисніть **«Відправити звіт»**, після чого повідомлення надійде у Discord.

> Вебхук можна змінити у файлі `script.js` (константа `webhookUrl`).
